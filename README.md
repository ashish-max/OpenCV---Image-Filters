# OpenCV---Image-Filters
 This project contains image filters applied to an image.<br>
 This is based on openCV and Python. 
# Available Filters:
1.Resizing<br>
2.Sharpening<br>
3.Gray-scale conversion<br>
4.Negative/inverse effect<br>
5.Gaussian Blur<br>
6.Pencil sketch<br>
7.Warming effect<br>
8.Cooling effect<br>
